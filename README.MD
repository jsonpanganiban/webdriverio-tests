WebdriverIO-mocha-jest-allure
================

Requirements
-----------------------
[Nodejs](https://nodejs.org/en/)

Getting Started
-----------------------
Install the dependencies
```
npm install
```

Execution
```
npx wdio wdio.conf.js
```

Reports
-------
Run below command to generate and open results
```
allure generate --clean allure-results
allure open
```
