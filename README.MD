WebdriverIO-mocha-jest-allure
================

Requirements
-----------------------
[Nodejs](https://nodejs.org/en/)

Getting Started
-----------------------
Install the dependencies
```
npm install
```

Execution
-----------------------
Specific Test/Spec
```
npm test -- --spec=tests/login.js
```
All Tests/Specs
```
npm test
```

Reports
-------
Run below command to generate and open results
```
npm run allure
```
